[GNOME] Καθυστέρηση Εκκίνησης Εφαρμογών

01 Mar 2018
Tips & Tricks
gnome / debian / ubuntu / desktop

Υπάρχουν πολλοί και διάφοροι τρόποι για να πετύχει κάποιος την καθυστέρηση εκκίνησης μιας εφαρμογής. Ίσως ο πιο συνηθισμένος από αυτούς να είναι η δημιουργία ενός απλού εκτελέσιμου script με sleep κλπ και η προσθήκη αυτού στις εφαρμογές εκκίνησης. Εδώ θα δούμε έναν απλό τρόπο, με μια γραμμή κώδικα την οποία προσθέτουμε στο ήδη υπάρχον αρχείο με κατάληξη .desktop Ο παρακάτω τρόπος θα λειτουργήσει στο γραφικό περιβάλλον GNOME και σε όσα αναγνωρίζουν τις παραμέτρους του (πχ Unity).
