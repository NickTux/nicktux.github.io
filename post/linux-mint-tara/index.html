Νέα χαρακτηριστικά στο Linux Mint 19

Tara, Cinammon Edition

24 Nov 2018
Linux / Linux Mint
release / new

  1. Η ταχύτερη αναζήτηση εφαρμογών
  2. Η προσθήκη υποστήριξης .flatpakref και .flatpakrepo

Το Linux Mint είναι υποστηρικτής περισσότερο των Flatpaks πακέτων, αντί των Snaps.

Σε αυτή τη δημοσίευση αναφέρθηκαν δυο από τα πολλά νέα χαρακτηριστικά του Mint 19. Τουλάχιστον αυτά που θεώρησα ως τα πιο σημαντικά, με το σημαντικότερο απ’ όλα την αλλαγή στην πολιτική των αναβαθμίσεων του συστήματος. Για να διαβάσετε όλα τα νεα χαρακτηριστικά, μεταβείτε στη σελίδα: New features in Linux Mint 19 Cinnamon
