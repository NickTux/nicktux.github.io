[GNOME] Καθυστέρηση Εκκίνησης Εφαρμογών

Καθυστέρηση εκκίνησης μιας εφαρμογής σε GNOME, μέσω desktop file.

01 Mar 2018
Ubuntu / Debian / Linux
tutorials / desktop / startup

Υπάρχουν πολλοί και διάφοροι τρόποι για να πετύχει κάποιος την καθυστέρηση εκκίνησης μιας εφαρμογής. Ίσως ο πιο συνηθισμένος από αυτούς να είναι η δημιουργία ενός απλού εκτελέσιμου script με sleep κλπ και η προσθήκη αυτού στις εφαρμογές εκκίνησης. Εδώ θα δούμε έναν απλό τρόπο, με μια γραμμή κώδικα την οποία προσθέτουμε στο ήδη υπάρχον αρχείο με κατάληξη .desktop.

Τοποθεσία Εφαρμογών Εκκίνησης

Τις βρίσκουμε συνήθως σε μια από τις παρακάτω τοποθεσίες

/etc/xdg/autostart #System Wide
~/.config/autostart #Current User

X-GNOME-Autostart-Delay

Βρίσκουμε το αρχείο εκκίνησης της εφαρμογής και ανοίγοντάς το με τον αγαπημένο μας editor (πχ. gedit) προσθέτουμε στο τέλος τη παρακάτω παράμετρο:

X-GNOME-Autostart-Delay=SECONDS

Αντικαθιστούμε το SECONDS με τα δευτερόλεπτα που θέλουμε να καθυστερήσει να εκκινήσει η εφαρμογή, από τη στιγμή σύνδεσης (login).
