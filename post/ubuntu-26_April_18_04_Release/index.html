[Ubuntu] 26 Απριλίου η νέα LTS έκδοση

Τι καινούριο υπάρχει στην έκδοση 18.04 LTS

Σε λίγες ημέρες από σήμερα η νέα έκδοση του Ubuntu 18.04 προβλέπεται να κυκλοφορήσει ως η νέα σταθερή LTS έκδοση.
Παρακάτω θα δούμε τι νέο φέρνει, πως μπορούμε να αναβαθμίσουμε με ασφάλεια και πότε, καθώς και άλλες χρήσιμες πληροφορίες.

Να υπενθυμίσουμε ότι οι εκδόσεις LTS (Long Term Support) υποστηρίζονται για 5 έτη από την ημερομηνία κυκλοφορίας τους, έτσι η 18.04 θα υποστηρίζεται με ενημερώσεις λογισμικού μέχρι και τον Απρίλιο του 2023.
Οι εκδόσεις LTS ανέκαθεν θεωρούταν(και τις περισσότερες φορές ήταν) πιο σταθερές από τις ενδιάμεσες(μη LTS), καθώς προορίζονται όχι μόνο για Home Users αλλά και για Enterprise/Business.
Αυτή τη στιγμή η 18.04 βρίσκεται σε Beta2 που σημαίνει ότι έχει μπει στη τελική ευθεία για κυκλοφορία. Σχεδόν τίποτα δεν θα αλλάξει σε σχέση με την τελική έκδοση, εκτός και αν αυτό χαρακτηριστεί ως Critical.

Γραφικό Περιβάλλον

Η μεγαλύτερη αλλαγή που έρχεται με την 18.04 LTS είναι σίγουρα το γραφικό περιβάλλον. Όπως οι περισσότεροι γνωρίζουμε αυτό πλέον έχει αλλάξει από το κλασσικό Unity στο GNOME, έκδοσης 3.28. Αν και έχει παραμετροποιηθεί έτσι ώστε να θυμίζει Unity. Έχει προστεθεί ένα Extention ονόματι Ubuntu Dock στα αριστερά το οποίο θυμίζει, αν και δεν είναι, τον Launcher του Unity. Γι’ αυτό το λόγο, αλλά και για άλλους, προσωπικά προτείνω τη νέα εγκατάσταση ως τρόπο αναβάθμισης σε αυτή την έκδοση και όχι την αναβάθμιση μέσω του Update Manager ή άλλων προγραμμάτων τρίτων.

Minimal Εγκατάσταση

Το πρόγραμμα εγκατάστασης της 18.04 προσφέρει μια νέα μέθοδο εγκατάστασης που ονομάζει Minimal. Από το όνομα και μόνο ο χρήστης καταλαβαίνει ότι πρόκειται για το “κανονικό” Ubuntu, χωρίς όμως πρόσθετες εφαρμογές. Περιέχει έναν Browser, έναν Text Editor, λίγα βασικά εργαλεία και το Software Center. Από εκεί και πέρα ο χρήστης μπορεί να επιλέξει τις εφαρμογές που θα κάνει εγκατάσταση και θα χρησιμοποιήσει. Κατά τη γνώμη μου ένα από τα καλύτερα χαρακτηριστικά αυτής της έκδοσης.

Προσθήκη αποθετηρίων PPA

Καιρός ήταν να αλλάξει και αυτό το χαρακτηριστικό. Η προσθήκη αποθετηρίων PPA από την 18.04 θα είναι λίγο ευκολότερη. Αυτό σημαίνει ότι χρησιμοποιώντας την εντολή sudo add-apt-repositroy η ενημέρωση των πηγών λογισμικού sudo apt-get update θα γίνεται αυτόματα. Έτσι ο χρήστης γλυτώνει ένα βήμα, μια εντολή, και μπορεί να προχωρήσει κατευθείαν στην εγκατάσταση του επιθυμητού πακέτου.

Πυρήνας Linux 4.15

Αν και οι συζητήσεις ήταν πολλές ως προς το ποια έκδοση πυρήνα θα χρησιμοποιηθεί σε αυτή την έκδοση, εν τέλει αποφασίστηκε για την 4.15.x. Η 4.14.x είναι αυτή τη στιγμή η επίσημη LTS έκδοση, από μεριάς Upstream Kernel, ωστόσο το Ubuntu διαθέτει μια ικανή ομάδα από δικούς του Kernel Developers/Maintainers οι οποίοι μπορούν να συντηρήσουν μια μη-LTS έκδοση με ενημερώσεις ασφαλείας ακόμη και όταν αυτή λήξει (upstream). Αυτό έχει συμβεί ξανά άλλωστε. Δεν είναι η πρώτη φορά.

Xorg Default

Και εδώ οι συζητήσεις ήταν πολλές και ενώ όλα έδειχναν τον Wayland ως Default Display Server, τελικά αποφασίστηκε ο παλιός αλλά σίγουρος και σταθερός Xorg. Και καλά έκαναν εδώ που τα λέμε, διότι ο Wayland αν και έχει φτάσει σε ένα πολύ καλό σημείο ανάπτυξης δεν είναι ακόμη έτοιμος να αντικαταστήσει τον Xorg σε ότι αφορά τη συμπεριφορά δημοφιλών εφαρμογών, ειδικά σε μια έκδοση που προορίζεται και για εταιρικούς πελάτες. Ο Wayland βέβαια υπάρχει και μπορεί να χρησιμοποιηθεί στο Ubuntu 18.04, μπορεί κάποιος να τον επιλέξει από τη Login Screen. Απλά δεν είναι ο προεπιλεγμένος.

Συλλογή δεδομένων χρήσης (opt-in)

Από αυτή την έκδοση του Ubuntu η Canonical αποφάσισε να συλλέγει δεδομένα χρήσης και hardware για λόγους ανάλυσης και χρήσης της διανομής. Τα δεδομένα αυτά θα αναρτούνται δημόσια και φυσικά δεν θα περιέχουν κανένα προσωπικό δεδομένο του χρήστη. Σχετικό FB Post.

Νέο πρόγραμμα εγκατάστασης Ubuntu Server

Από αυτή την έκδοση ένα νέο πρόγραμμα εγκατάστασης που αφορά μόνο τις Server εκδόσεις θα είναι διαθέσιμο. Σίγουρα είναι πιο όμορφο και θα διευκολύνει ακόμη περισσότερο τον νεοεισερχόμενο να κάνει εγκατάσταση το Ubuntu Server 18.04. Σχετικό FB Post.

Νέο theme και icon set. Όχι ακόμη.

Αν και έχει προταθεί και έχει γίνει αποδεκτό, το νέο Theme και σετ εικονιδίων το οποίο έχει δημιουργηθεί από την κοινότητα δεν πρόλαβε να ενσωματωθεί στη 18.04 LTS καθώς δεν θεωρείται ακόμη έτοιμο.
Σε όλες τις εκδόσεις του Ubuntu, χρόνια τώρα, επικρατεί ένα και μόνο θέμα, το Ambiance. Ήρθε ο καιρός να αλλάξει και αυτό. Η πρόταση έγινε αποδεκτή και πλήθως χρηστών συνεισφέρουν στο νέο Θέμα και Εικονίδια για το Ubuntu. Για λεπτομέρειες δείτε στο Ubuntu Community. Το νέο Θέμα και Εικονίδια αναμένεται πιθανότατα στη 18.10.

Με γυμνό μάτι

Άλλες διαφορές που θα εντοπίσει ένας χρήστης άμεσα είναι το νέο Wallpaper, καθώς και ο Nautilus File Manager (Files). Αν και ο τελευταίος θα μείνει στην έκδοση 3.26.

Γνωστά σφάλματα (Bugs)

Σφάλματα τα οποία έχουν αναφερθεί μπορούν να βρεθούν σε συγκεκριμένη σελίδα στο Launchpad. Δείτε εδώ. Από όλα αυτά τα σφάλματα βέβαια θα γίνει εκαθάριση των duplicates, των λανθασμένων και εκείνα που πιθανόν να μη διοθρωθούν ποτέ. Οι διοθρώσεις σφαλμάτων ίσως να μη προλάβουν να γίνουν με τη κυκλοφορία της έκδοσης 18.04, αλλά στο πρώτο Point Release.
Σε κάθε περίπτωση δείτε τις σημειώσεις έκδοσης (Release Notes). Εκεί αναφέρονται τα πιο γνωστά σφάλματα τα οποία επηρεάζουν συνολικά το λειτουργικό σύστημα.
Ένα που θέλω να αναφέρω εδώ είναι κάτι που εντόπισα πρόσφατα (μόλις χθές) και κατά τη γνώμη μου είναι σημαντικό. Δείτε την αναφορά σε αυτό το Link και αν σας επηρεάζει πατήστε το κουμπί “Yes it affects me” ώστε να ανέβει το Heat και να λάβει τη δέουσα προσοχή.

Αναβάθμιση ή Νέα Εγκατάσταση;

Κι εδώ ερχόμαστε στα γνωστά ερωτήματα που απασχολούν σε κάθε νεά έκδοση. Να αναβαθμίσω ή όχι; Αν ναι, πότε; Και αν ναι, πως; Κάποιος κανόνας δεν υπάρχει, γι’ αυτό σε αυτή τη παράγραφο γράφω τη γνώμη μου και μόνο.

Αν είστε Οικιακός χρήστης (Home User) να αναβαθμίσετε στη νέα έκδοση. Ο τρόπος είναι δική σας επιλογή αλλά προσωπικά προτείνω τη νέα εγκατάσταση και αυτό λόγω της μεγάλης αλλαγής που φέρνει η 18.04, την αλλαγή γραφικού περιβάλλοντος. Βέβαια το πρόγραμμα αναβάθμισης λειτουργεί και θεωρητικά έχει ελεγχθεί. Θυμηθείτε όμως, ή οι νεότεροι μάθετε, ότι παλαιότερα κάποιες διανομές πρόσφεραν τη νέα εγκατάσταση ως τον μοναδικό τρόπο αναβάθμισης από μια έκδοση σε μια άλλη, και λειτουργούσε μια χαρά. Αυτό που έχει να κάνει ο χρήστης είναι να αποθηκεύσει τα προσωπικά του αρχεία σε ένα άλλο δίσκο και να τα επαναφέρει ξανά μετά την ολοκλήρωση της νέας εγκατάστασης.
Η αναβάθμιση μπορεί να επιτευχθεί και μέσω του προγράμματος εγκατάστασης του Ubuntu. Δηλαδή, θα εμφανιστεί εκεί η κατάλληλη επιλογή όταν το πρόγραμμα εγκατάστασης της 18.04 ανιχνεύσει παλαιότερη έκδοση.

Αν είστε εταιρικός χρήστης (Business/Enterprise) ή ακόμη και αν είστε Home User που δεν σας αρέσουν οι εκπλήξεις, δεν το συζητάμε καν. Μένουμε στη 16.04 LTS μέχρι να μας ειδοποιήσει ο Update Manager για τη διαθεσιμότητα νέας έκδοσης, χωρίς να τον “πιέσουμε”. Αυτό θα γίνει εφόσον έχουμε ενεργοποιημένη την επιλογή “ειδοποίηση για νέες εκδόσεις” και θα γίνει με τη κυκλοφορία του πρώτου Point Release (18.04.1). Τότε θεωρείται η διανομή σταθερή και με τα περισσότερα Critical/High Bugs διορθωμένα. Επίσης έχει δοκιμαστεί από πλήθος άλλων χρηστών οι οποίοι σπεύδουν να αναβαθμίσουν από την Ημέρα 0, οπότε παίρνουμε κάτι έτοιμο, δοκιμασμένο και σταθερότερο.

Αν δεν θέλετε να κάνετε αναβάθμιση δεν υπάρχει κάποιο πρόβλημα σε ότι αφορά την υποστήριξη και τις ενημερώσεις. Αυτό ισχύει για όσους/ες διαθέτουν τη 16.04 LTS η οποία θα υποστηρίζεται έως τον Απρίλιο του 2021. Αν όμως έρχεστε από μια μη-LTS έκδοση, τότε σχεδόν επιβάλλεται να αναβαθμίσετε μιας και η υποστήριξη θα λήξει σχετικά γρήγορα. Η 17.10 θα υποστηρίζεται έως τον Ιούλιο 2018.
