Προβλήματα στην ασύρματη σύνδεση

σε λειτουργικό σύστημα Ubuntu/Linux

02 Jul 2013
Linux / Ubuntu
troubleshooting / wireless 
ΑΥΤΗ Η ΔΗΜΟΣΙΕΥΣΗ ΕΧΕΙ ΓΙΝΕΙ ΠΑΝΩ ΑΠΟ ΕΝΑ ΧΡΟΝΟ ΠΡΙΝ! ΜΠΟΡΕΙ ΝΑ ΜΗΝ ΙΣΧΥΕΙ ΠΛΕΟΝ.

Τα προβλήματα στην ασύρματη σύνδεση μπορεί να ποικίλουν. Γι’ αυτό δεν υπάρχει κάποιος γενικός κανόνας επίλυσης τέτοιων προβλημάτων. Το καλύτερο που μπορείτε να κάνετε είναι να ζητήσετε βοήθεια από ένα φόρουμ υποστήριξης και βοηθείας για το Ubuntu.

Παρακάτω θα παραθέσω κάποιες εντολές διάγνωσης. Οι εντολές αυτές περιέχονται σε ένα script , το οποίο έχει γραφτεί από χρήστες του UbuntuForums.org και τα αποτελέσματα θα βοηθήσουν αρκετά όσους ασχολούνται με wireless troubleshooting.

Η παρακάτω διαδικασία απαιτεί ενεργή σύνδεση στο Internet.

Συνδέστε τον υπολογιστή σας στο Internet μέσω καλωδίου Ethernet .

Ανοίξτε ένα τερματικό με τον συνδυασμό πλήκτρων CTRL+ALT+T και εκτελέστε τις παρακάτω εντολές μια-μια.

wget -N -t 5 -T 10 "https://dl.dropboxusercontent.com/s/l93ifv397doyvil/wireless-script"
chmod +x wireless-script
./wireless-script

Οι παραπάνω εντολές θα κατεβάσουν και θα εκτελέσουν το script ονόματι wireless-script.

Όταν εκτελεστεί η τελευταία εντολή θα δημιουργηθεί ένα αρχείο ονόματι wireless-info.txt στον προσωπικό σας κατάλογο. Δώσε τον κωδικό σας όταν σας ζητηθεί.

Αυτό το αρχείο περιέχει όλες τις πληροφορίες που χρειάζεται ένας troubleshooter για να προσπαθήσει να λύσει το πρόβλημα σας. Μπορείτε να κάνετε αντιγραφή και επικόλληση όλα τα περιεχόμενα σε ένα φόρουμ υποστήριξης και βοήθειας για το Ubuntu ή σε κάποια υπηρεσία Pastebin.

Σε περίπτωση που το script ανιχνεύσει το πρόγραμμα pastebinit εγκατεστημένο, θα σας προτείνει να ανεβάσει το περιεχόμενο στο προεπιλεγμένο paste site που είναι το http://paste.ubuntu.com/.

Σημείωση: Όλα τα προσωπικά σας στοιχεία , όπως access point , ESSID κλπ , έχουν αφαιρεθεί από τα αποτελέσματα του script για λόγους ασφαλείας.

Το αρχείο wireless-info.txt που περιέχει τα αποτελέσματα μπορείτε να το ανοίξετε με έναν κειμενογράφο. Στο Ubuntu ως προ-εγκατεστημένος κειμενογράφος είναι το gedit , οπότε αρκεί να δώσετε στο τερματικό σας την εντολή

gedit wireless-info.txt

ή απλά να κάνετε ένα δεξί κλικ στο αρχείο wireless-info.txt και να πατήσετε άνοιγμα (θα ανοίξει με τον προ-επιλεγμένο κειμενογράφο) Έτσι θα μπορέσετε να δείτε και να κάνετε αντιγραφή - επικόλληση τα αποτελέσματα.

Αν θέλετε να δείτε τα περιεχόμενα του script, τις εντολές που περιέχει, ανοίξτε το αρχείο (wireless-script) με έναν κειμενογράφο

H παρακάτω διαδικασία δεν απαιτεί σύνδεση στο Internet.

Σε περίπτωση που δεν έχετε την δυνατότητα ενσύρματης σύνδεσης στο Internet από το λειτουργικό σύστημα Ubuntu , τότε μπείτε από έναν υπολογιστή που διαθέτει σύνδεση στο Internet και κάνετε κλικ εδώ . Πατήστε “Save page as…” , έτσι ώστε να κατέβει και να σωθεί το αρχείο με την ονομασία wireless-script. Έπειτα πρέπει να το μεταφέρετε (π.χ μέσω USB-stick) στο λειτουργικό σύστημα Ubuntu, το οποίο «πάσχει» από το πρόβλημα. Αφού το μεταφέρετε κάνετε δεξί κλικ στο αρχείο (wireless-script) και κλικ στο properties (Ιδιότητες) , στο παράθυρο που θα ανοίξει πάτε στην καρτέλα Permissions(δικαιώματα) και μαρκάρετε το κουτάκι “Allow executing file as program” (να επιτρέπεται η εκτέλεση του αρχείου ως πρόγραμμα).

Τέλος κάνετε ένα διπλό κλικ επάνω στο αρχείο wireless-script και πατήστε Run (Εκτέλεση).

Έπειτα ισχύει η ίδια διαδικασία (όπως παραπάνω) προκειμένου να ανοίξετε και να δείτε τα περιεχόμενα του αρχείου wireless-info.txt που θα δημιουργηθεί.
