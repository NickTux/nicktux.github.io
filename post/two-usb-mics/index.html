Δύο USB μικρόφωνα σε Linux

και τι να προσέξουμε

05 Apr 2019
Ubuntu / Linux Mint
mic / podcast / usb

Καλή Χρονιά είπαμε;
Νομίζω είναι η πρώτη δημοσίευση του 2019. Ξέρετε, επειδή δεν είμαι επαγγελματίας αρθρογράφος και επειδή είμαι ένθερμος υποστηρικτής της άποψης “Γράψε, όταν έχεις κάτι να γράψεις”, επιπροσθέτως ότι ο ελεύθερος χρόνος είναι πλέον λίγος, το αποτέλεσμα είναι ακριβώς αυτό.
Αυτή τη φορά έχω να δημοσιεύσω κάτι όμως, το οποίο ίσως βοηθήσει άλλους που θα ήθελαν να πάρουν μια απόφαση στην αγορά ξεχωριστού/αποκλειστικού μικροφώνου πχ. για podcasts, youtube, live chatting, gaming κλπ.

Η λάθος επιλογή

Θα προτείνω να ψάξετε αρκετά πριν αποφασίσετε τι είδους μικρόφωνο θα αγοράσετε. Σε γενικές γραμμές υπάρχουν δυο είδη μικροφώνων, τα πυκνωτικά (condenser) και τα δυναμικά (dynamic). Υπάρχουν κι άλλα πολλά χαρακτηριστικά τα οποία όμως θεωρώ δευτερεύουσας σημασίας για τη παρούσα δημοσίευση. Σε ότι αφορά τη σύνδεση, τα XLR απαιτούν έξτρα εξοπλισμό για σύνδεση με υπολογιστή. Τα USB φυσικά δεν έχουν τέτοιες απαιτήσεις, απλά τα συνδέεις και (τις περισσότερες φορές) απλά δουλεύουν.
Αν και το έψαξα αρκετά, με reviews στο Youtube, άρθρα κλπ, κατέληξα να κάνω τη λάθος (για μένα) επιλογή. Αυτή ήταν ένα Samson Meteor το οποίο είναι πραγματικά ένα εξαιρετικό μικρόφωνο, αλλά δεν είναι για μένα ή για να το πω καλύτερα για το χώρο που διαθέτω.
Βλέπετε, τα πυκνωτικά μικρόφωνα (όπως το Meteor) είναι αρκετά ευαίσθητα στη “σύλληψη” ήχων από παντού. Το συγκεριμένο είναι μεν καρδιοειδές (cardioid), αλλά αυτό λίγη σημασία έχει στη δική μου περίπτωση. Δηλαδή, μιλάμε ότι αν περάσει κουνούπι εκείνη την ώρα θα ακουστεί στο μικρόφωνο. Αυτό για άλλους είναι καλό, για άλλους (σαν εμένα) είναι κακό.
Το αποτέλεσμα της παραπάνω λάθους επιλογής ήταν να παραγγείλω άλλο ένα μικρόφωνο, αυτή τη φορά δυναμικό. Το αρνητικό με αυτές τις αγορές είναι ότι δύσκολα μπορεί κάποιος να δοκιμάσει το προϊόν πριν το αγοράσει.
Να σημειώσω εδώ ότι τα συγκεκριμένα δυο μικρόφωνα δεν υπήρχαν διαθέσιμα στην Ελληνική αγορά, τουλάχιστον σε ακτίνα 10χλμ από το σπίτι μου.
Επαναλαμβάνω ότι το μικρόφωνο είναι εξαιρετικό και ως κατασκευή και ως απόδοση. Απλά δεν είναι για μένα. Δείτε σχετικό review με δοκιμή στο παρακάτω βίντεο. Λάβετε υπόψη σας ότι ο τύπος έχει, όπως θα δείτε και μόνοι σας, (σχεδόν) studio στο σπίτι του.

Η σωστή επιλογή

Η σωστή επιλογή για μένα, αυτή που μου ταίριαξε καλύτερα, ήταν ένα Samson Q2U Dynamic. Άλλη κατασκευή βέβαια, στιβαρή και αυτή μεν, αλλά διαφορετικής “νοοτροπίας”. Είμαι έτοιμος για πίστα λέμε.
Χρειάζεται να είσαι πιο κοντά στο μικρόφωνο για να γράψεις σωστά, ενώ στο πυκνωτικό αυτό είναι πιο δύσκολο (μπουκώνει). Για τη “δουλειά” που το θέλω εγώ είναι μια χαρά. Το προηγούμενο απλά ήταν “too much”. Δείτε review του Q2U στο παρακάτω βίντεο.

Audacity

Όσοι ασχοληθείτε με το “άθλημα” θα προτείνω ένα πρόγραμμα ανοιχτού κώδικα με το όνομα Audacity. Θέλει μια μικρή εξοικείωση για να το δουλεύετε άνετα, αλλά μικρή. Δεν είναι Blender ή Kdenlive και έχει εξίσου καλά αποτελέσματα, σε ότι αφορά βέβαια τον ήχο που επεξεργάζεστε. Δείτε ένα βίντεο για νέους χρήστες (από τα πολλά που κυκλοφορούν εκεί έξω) που αφορά το Audacity.

Windows & Linux

Ωραία όλα τα παραπάνω, ας πάμε όμως στο “ζουμί” της υπόθεσης. Λοιπόν, απλά και απέριττα θα σας πω ότι και τα δυο μικρόφωνα λειτουργούν καλύτερα σε Ubuntu/Linux 18.04 και Linux Mint 19.1 απ’ ότι σε Windows 10. Η διαφορά εδώ είναι ένα πρόγραμμα (Samson Sound Deck), το οποίο προσωπικά θεωρώ απαραίτητο, που πρέπει να κάνεις εγκατάσταση σε Windows ώστε να μπορείς να γράψεις σαν άνθρωπος. Αυτό το πρόγραμμα βρίσκεται εδώ.
Σε Linux, από την άλλη, δεν χρειάζεται κάτι τέτοιο (και να θέλαμε δεν υπάρχει). Ο θόρυβος (background noise) είναι λιγότερος (στα δικά μου αυτιά) και διορθώνεται εύκολα με το Audacity. Να πούμε εδώ για όσους δεν το γνωρίζουν ότι το Audacity είναι διαθέσιμο και σε Windows, αλλά για μένα δεν ήταν αρκετό για να μειώσω σε ικανοποιητικό επίπεδο τον θόρυβο (λαμβανομένου υπόψη ότι δεν είμαι και κανένας expert στο Audacity).
Επίσης το Monitoring, είναι η διαδικασία αυτή που ακούς σε realtime χωρίς καθυστερήσεις (latency) τι λες και τι γράφεις, λειτουργεί επίσης απρόσκοπτα (και στα δυο μικρόφωνα).
