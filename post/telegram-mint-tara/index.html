Εγκατάσταση του Telegram Desktop σε Linux Mint

Tara, Cinammon Edition

25 Nov 2018
Linux / Linux Mint
telegram / apps / packages / tutorials

Σε αυτή τη δημοσίευση θα δούμε βήμα-βήμα πως κάνουμε λήψη και εγκατάσταση του Telegram Desktop App, στο λειτουργικό μας σύστημα.
Αν και φαντάζει απλό και εύκολο, θα κάνουμε τη διαδικασία με έναν διαφορετικό τρόπο και θα εξηγήσουμε το (τα) γιατί.

To να κάνουμε εγκατάσταση πακέτα/προγράμματα/εφαρμογές οι οποίες έχουν ελεγχθεί και φιλοξενούνται στα επίσημα αποθετήρια της διανομής που χρησιμοποιούμε, είναι το σωστό. Αυτά τα πακέτα συνήθως τα βρίσκουμε στο(ν) Software Center/Manager. Είναι η πιο ασφαλής μέθοδος και εκτός αυτού αποφεύγουμε μπελάδες όπως ανικανοποίητες εξαρτήσεις κλπ.
Για να παραλείψετε τις επεξηγήσεις και να πάτε κατευθείαν στην εγκατάσταση, κλικ εδώ.

Υπάρχουν όμως λίγες περιπτώσεις που η εγκατάσταση από το Official Site της εφαρμογής, δηλαδή με τον ΜΗ-προτεινόμενο τρόπο, είναι σε κάποια σημεία καλύτερη. Μια από αυτές τις περιπτώσεις είναι το Telegram και παρακάτω δείχνουμε το γιατί (σύμφωνα με τη προσωπική γνώμη του συντάκτη).

Ανοίγουμε τον διαχειριστή λογισμικού και ψάχνουμε για το Telegram. Η αναζήτηση θα επιστρέψει δυο αποτελέσματα.


Ωραία. Ενστικτωδώς πηγαίνουμε στη πρώτη εφαρμογή της λίστας μιας και έχει βαθμολογία 4.2 και κάνοντας κλικ στο Install Button, βλέπουμε τη παρακάτω εικόνα.


Εξωφρενικό ;
Μια εφαρμογή σαν το Telegram, θα καταλάβει 788,1 ΜΒ , ενώ ζητάει τουλάχιστον 2,5 GB χώρο στο δίσκο. Μετά αναρωτιόμαστε γιατί δεν προχωράνε τα Flatpaks και δεν τα προτιμάει ο κόσμος. Θα τα προτιμήσει, όταν οι SSD δίσκοι του 1TB+ θα κοστίζουν 100€.
Το παραπάνω, για ‘μένα, είναι αρκετό για να προχωρήσω στη δεύτερη στη λίστα εφαρμογή. Εκεί, κάνοντας κλικ στο Install Button βλέπουμε το παρακάτω.


Normal μέγεθος εφαρμογής, μόλις 30,3 MB. Τι γίνεται όμως με την έκδοση; Βλέπω αναφέρει την 2.17. Είναι άραγε αυτή η τελευταία έκδοση του Telegram; και αν όχι, πόσο παλιά είναι;

Πηγαίνοντας στο Official Site και κάνοντας κλικ στο Download βλέπουμε …

Από την 2.17 στην 4.3 φαντάζομαι μεσολαβούν κάμποσες εκδόσεις. Αναζητώντας επιβεβαίωση πάμε στο Launchpad, μιας και το πακέτο αυτό αναπτύσσεται και φιλοξενείται στο universe αποθετήριο του Ubuntu 18.04.


Το βρήκαμε. Αυτή είναι η έκδοση 2.17 λοιπόν, η οποία μεταφορτώθηκε στο αποθετήριο τον Απρίλιο του 2018. Να υπενθυμίσω ότι κοντεύει να φύγει το 2018.

Αυτόματα έρχεται στο μυαλό κάποιου χρήστη, ας το πούμε λίγο εξοικειωμένου με το Ubuntu/Mint, να ψάξει σε κάποιο προσωπικό αποθετήριο (PPA). Υπάρχει κάτι άραγε; κάτι Official όμως. Όχι του Νίκου, του Μήτσου ή του Κίτσου.
Ψάχνοντας με “Official Telegram PPA Ubuntu” εντόπισα το παρακάτω, το οποίο για την έκδοση Bionic (στην οποία βασίζεται το Tara) δεν βλέπω κάτι αξιόλογο.


Έχουμε κάτι άλλο; ( πριν πάμε να τρέξουμε την έκδοση από το Website; ) ας σημειώσουμε εδώ ότι πρόκειται για μια εφαρμογή Ανοιχτού Κώδικα.
Α, ναι. Έχουμε και τα Snaps. Αν και στο Mint η βάση δεν είναι καν εγκατεστημένη, αυτό είναι εύκολο. Δίνουμε σε ένα τερματικό τη παρακάτω εντολή:

sudo apt install snapd

και είμαστε έτοιμοι.

Τι έκδοση φιλοξενείται εκεί και πόσα MB είναι άραγε;


Πολύ καλύτερα από το Flatpak. Πάλι σε σχέση με την εφαρμογή στο Website είναι μεγαλύτερο σε όγκο, αλλά “τρώγεται”. Και η έκδοση είναι η τελευταία.


Συμπεραίνουμε ότι, αν θέλουμε την τελευταία έκδοση του Telegram, η οποία να μη πιάσει περί τα 1000ΜΒ στο δίσκο μας, έχουμε δυο επιλογές. Website ή Snap. Θα δούμε την εγκατάσταση και με τους δυο τρόπους, κλείνοντας προς εκείνον του Website λόγω μικρότερου όγκου της εγκατάστασης.

Snap

Μέσω τερματικού εκτελούμε τις παρακάτω εντολές μια-μια

sudo apt install snapd
sudo snap install telegram-desktop

Website

Κάνουμε λήψη το αρχείο από την Official σελίδα. Ανοίγουμε τον Nemo (ή όποιον διαχειριστή αρχείων έχουμε) βρίσκουμε το αρχείο και με δεξί κλικ κάνουμε αποσυμπίεση (extract here).
Στον αποσυμπιεσμένο φάκελο, ο οποίος έχει το όνομα Telegram, θα βρούμε δυο εκτελέσιμα αρχεία. Το ένα ονομάζεται Telegram και το άλλο Updater.


Με διπλό κλικ στο Telegram θα ανοίξει η εφαρμογή και είμαστε έτοιμοι να τη χρησιμοποιήσουμε. Τόσο απλά!


Είδαμε λοιπόν ότι στη περίπτωση του Telegram, η καλύτερη (ίσως) αλλά σίγουρα η πιο συμφέρουσα λύση είναι η λήψη της εφαρμογής από την Official σελίδα. Αν και στο Linux γενικότερα αυτό είναι καλό να το αποφεύγουμε υπάρχουν περιπτώσεις σαν κι αυτή που μπορούμε να “παρακάμψουμε” τους προτεινόμενους τρόπους, και με το δίκιο μας.

Μη ξεχνάμε και το κανάλι επικοινωνίας της ελληνόφωνης κοινότητας του Linux Mint στο Telegram. Τώρα που κάναμε εγκατάσταση την εφαρμογή, το μόνο που έχουμε να κάνουμε είναι Join και να πούμε ένα γεια. (εχμ, το “να πούμε ένα γεια” δεν το λέω έτσι. Σχεδόν επιβάλλεται. Διαφορετικά μπορεί να σας θεωρήσουν Bot και να φάτε Ban)

Desktop Launcher;

Πριν κλείσουμε να πούμε ότι το εκτελέσιμο αρχείο δημιουργεί και Desktop Launcher με Exec= την τοποθεσία από την οποία έτρεξε για πρώτη φορά.
