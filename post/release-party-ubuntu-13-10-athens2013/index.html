Release Party Ubuntu 13.10 Athens 2013

02 Jul 2013
Ubuntu / Linux
desktop / events / over_a_year_old

To release party για το Ubuntu 13.04 διεξήχθη με επιτυχία και φέτος στο καφέ Ηλιοστάσιο στο Θησείο. Ο κόσμος προσήλθε σε ικανοποιητικό αριθμό, η τούρτα όμορφη και εύγευστη και το κλίμα όπως πάντα φιλικό.

 

Περισσότερο φωτογραφικό υλικό στο φώτο-άλμπουμ της Ελληνικής κοινότητας.

Until next time ... "Keep Your Mind in a Constant State of Openness"
