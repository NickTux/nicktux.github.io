Επαναφορά κωδικού χρήστη

σε Ubuntu/Linux

02 Jul 2013
Ubuntu / Linux
tutorials / troubleshooting / password / over_a_year_old

Υπάρχουν διάφοροι λόγοι όπου μπορεί να θέλουμε να επαναφέρουμε τον κωδικό μας στο Ubuntu.

  1. Ξεχάσαμε τον κωδικό μας
  2. Θέσαμε κατά λάθος έναν κενό κωδικό και τώρα δεν μπορούμε να εγκαταστήσουμε τίποτα
  3. Αλλάξαμε τον κωδικό μας και πλέον δεν τον θυμόμαστε
ΑΥΤΗ Η ΔΗΜΟΣΙΕΥΣΗ ΕΧΕΙ ΓΙΝΕΙ ΠΑΝΩ ΑΠΟ ΕΝΑ ΧΡΟΝΟ ΠΡΙΝ! ΜΠΟΡΕΙ ΝΑ ΜΗΝ ΙΣΧΥΕΙ ΠΛΕΟΝ.

Σε τέτοιες περιπτώσεις μπορούμε να ακολουθήσουμε την παρακάτω διαδικασία για να κάνουμε ένα reset (προσέξτε , δεν είναι επαναφορά του παλιού κωδικού , είναι reset) τον κωδικό μας στο Ubuntu. Δηλαδή να θέσουμε έναν καινούριο κωδικό που να τον θυμόμαστε.

Αν έχετε διπλό σύστημα εκκίνησης (dual-boot) τότε θα βλέπετε το μενού του grub-bootloader κατά την εκκίνηση του υπολογιστή σας. Αν έχετε μόνον Ubuntu , τότε θα πρέπει να κρατήσετε πατημένο το πλήκτρο [Shift] για να εμφανιστεί το μενού.

Επιλέξτε την εισαγωγή με το (Recovery mode)[Λειτουργία ανάκτησης].

Σημείωση: Εάν έχετε έκδοση Ubuntu 12.10 και νεότερη , τότε η παραπάνω εικόνα ενδέχεται να αλλάζει λίγο. Θα πρέπει να επιλέξετε πρώτα Advanced options for Ubuntu και μετά θα δείτε την επιλογή Recovery mode.

Έπειτα θα ανοίξει μια άλλη οθόνη με διάφορες επιλογές,

πρέπει να επιλέξετε root (Drop to root shell prompt).

Έπειτα πρέπει να δώσετε κάποιες εντολές για να γίνει ο κωδικός σας reset.

Πρώτα δίνετε

mount -o rw,remount

για να προσαρτηθεί το σύστημα αρχείων σε Read-Write. Η παραπάνω εντολή , αν εκτελεστεί σωστά , δεν πρέπει να σας επιστρέψει κάποιο αποτέλεσμα.

Η επόμενη εντολή είναι που θα κάνει reset τον κωδικό σας , βασιζόμενη στο username που έχετε . Οπότε αν το username σας είναι π.χ nikos , τότε δίνετε

passwd nikos

Θα σας ζητήσει δυο φορές να γράψετε τον νέο κωδικό. Προσέξτε διότι δεν θα φαίνεται κάτι. Ούτε αστεράκια , ούτε τίποτα. Σαν να μην γράφετε καθόλου. Αυτό είναι φυσιολογικό. Αφού δώσετε δυο φορές τον νέο πλέον κωδικό και τον δεχτεί , είσαστε έτοιμοι να κάνετε ένα reboot και να μπείτε στο Λειτουργικό σύστημα με τον νέο κωδικό. Δώστε την εντολή

reboot

για να γίνει επανεκκίνηση.

Σε περίπτωση που δεν θυμάστε ούτε το όνομα χρήστη , μπορείτε να δώσετε την εντολή

ls /home

και θα σας επιστρέψει το όνομα χρήστη (ή ονόματα χρηστών αν είναι παραπάνω από ένας).

Η παραπάνω διαδικασία ισχύει για όλα τα λειτουργικά συστήματα Ubuntu-based (Ubuntu , Xubuntu , Lubuntu , Kubuntu , Linux mint … κλπ).

Παρακάτω μπορείτε να δείτε ένα βίντεο επαναφοράς κωδικού από το κανάλι της Ελληνικής κοινότητας Ubuntu στο youtube.
