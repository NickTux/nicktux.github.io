Επιδόρθωση στο διαχειριστή πακέτων

Ubuntu/Linux

02 Jul 2013
Ubuntu / Linux
tutorials / troubleshooting / package manager / over_a_year_old

Όταν λέμε διαχειριστής πακέτων , εννοούμε οτιδήποτε έχει σχέση με την εγκατάσταση ή αναβάθμιση των πακέτων - προγραμμάτων του Ubuntu. Το πρόβλημα μπορεί να είναι , είτε “μη επιλυμένες εξαρτήσεις” (dependencies) , είτε αδυναμία αναβάθμισης , είτε κρατούμενα ελαττωματικά πακέτα … κλπ.

ΑΥΤΗ Η ΔΗΜΟΣΙΕΥΣΗ ΕΧΕΙ ΓΙΝΕΙ ΠΑΝΩ ΑΠΟ ΕΝΑ ΧΡΟΝΟ ΠΡΙΝ! ΜΠΟΡΕΙ ΝΑ ΜΗΝ ΙΣΧΥΕΙ ΠΛΕΟΝ.

Σε τέτοιες περιπτώσεις είναι καλό να κάνουμε μόνοι μας μια διάγνωση ή και προσπάθεια διόρθωσης , πριν ζητήσουμε βοήθεια από κάποιο φόρουμ.. κλπ. Έτσι σε περίπτωση μη επίλυσης του προβλήματος θα έχουμε τα αποτελέσματα της διαδικασίας που ακολουθήσαμε έτοιμα και διαθέσιμα για να τα κάνουμε ποστ. Τα αποτελέσματα της παρακάτω διαδικασίας είναι σημαντικά και εκτός του ότι η ίδια η διαδικασία μπορεί αυτόματα να λύσει το πρόβλημα σας , θα σας παρέχει και διαγνωστικές πληροφορίες σχετικά με το τι συμβαίνει.

Αυτόματο Script

Ανοίξτε ένα τερματικό (CTRL+ALT+T) και εκτελέστε τις παρακάτω εντολές με τη σειρά , μια-μια

wget "https://dl.dropboxusercontent.com/s/w8nw83qjw2khl08/fixpackage" -O ~/fixpackage
chmod +x fixpackage
sudo ./fixpackage

Το παραπάνω script , ονόματι fixpackage , δεν είναι τίποτε άλλο από μια σειρά εντολών. Αντί να τις κάνετε copy-paste μια μια (είναι και πολλές) , απλά τρέχετε το script  και έτσι δεν υπάρχει και περίπτωση να κάνετε κάποιο λάθος ή να παραλείψετε κάποια εντολή. Αν θέλετε να δείτε τα περιεχόμενα του script , τρέξτε την εντολή

paste fixpackage

Σημαντικό : Ρυθμίστε το τερματικό σας έτσι ώστε η εκτύπωση γραμμών να είναι απεριόριστη. Αφού ανοίξετε το τερματικό , πατήστε Edit → Profile Preferences . Στο παράθυρο που θα ανοίξει πηγαίνετε στο Scrolling και τικάρετε το κουτάκι που γράφει Unlimited. Έτσι το τερματικό δεν θα κόψει αποτελέσματα εντολών.

Αφού τρέξετε το παραπάνω script , πιθανών το πρόβλημα που είχατε να έχει διορθωθεί ήδη. Αν όχι , τότε όλα τα αποτελέσματα που έβγαλε στο τερματικό είναι διαγνωστικά για όσους λύνουν προβλήματα στο Ubuntu. Να τα παραθέσετε μαζί με το αίτημα σας για βοήθεια. Μια αντιγραφή - επικόλληση όλων των αποτελεσμάτων είναι αρκετή.
