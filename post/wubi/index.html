Wubi - Ένα παρωχημένο project

Που δεν πρέπει πλέον να χρησιμοποιείτε.

11 Jun 2013
Ubuntu / Linux
announce / windows installer / wubi / over_a_year_old 
ΑΥΤΗ Η ΔΗΜΟΣΙΕΥΣΗ ΕΧΕΙ ΓΙΝΕΙ ΠΑΝΩ ΑΠΟ ΕΝΑ ΧΡΟΝΟ ΠΡΙΝ!

Το wubi.exe (Windows Ubuntu Installer) όταν ξεκίνησε ήταν ένα από τα πιο δημοφιλή projects ου Ubuntu.

Σου επέτρεπε να εγκαταστήσεις το Ubuntu μέσα στα Windows και να το δεις, να δουλέψεις σε αυτό. Μέσω αυτού του project το Ubuntu απέκτησε λίγο παραπάνω φήμη και αναγνώριση από εκείνη που είχε τότε. Οι χρήστες του windows λειτουργικού μπορούσαν εύκολα και γρήγορα να εγκαταστήσουν το Ubuntu μέσω του wubi.exe και μετά από μια επανεκκίνηση να δουν το Ubuntu να παρατίθεται στην λίστα του windows bootloader.

Τι κερδίζετε

Αυτό που κέρδιζαν οι χρήστες του wubi.exe ήταν η γρήγορη και απλή εγκατάσταση του Ubuntu. Όταν πρώτο ξεκίνησα με το Ubuntu ήμουν κι εγώ ένας από εκείνους που υποστήριζαν αυτό το project. Μια χαρά μου φαινόταν. Μέχρι που μια «ωραία πρωία» έγιναν όλα μπάχαλο. (και για του λόγου το αληθές). Αυτό που κερδίζουν όσοι χρησιμοποιούν wubi, είναι πολύ λίγο σε σχέση με αυτά που χάνουν.

Τι χάνετε

Καταρχήν χάνετε σε ταχύτητα - απόκριση συστήματος - και αποφεύγετε με κάποιον τρόπο να μπείτε στην λογική του Linux. Να έχετε υπόψιν σας πως δεν πρόκειται για μια κανονική εγκατάσταση Linux. Με το wubi εγκαθιστάτε το Ubuntu ως ένα απλό πρόγραμμα windows. Θα το βρείτε-δείτε και στην προσθαφαίρεση προγραμμάτων στον πίνακα ελέγχου των windows. Επίσης υπάρχει περίπτωση, ειδικά αν χρησιμοποιήσετε προγράμματα τύπου EasyBCD, να καταστρέψετε τον bootloader των windows και να μην μπορείτε στο τέλος να κάνετε boot ούτε σε αυτά. Έχετε περιορισμό στον χώρο. Το wubi.exe επιτρέπει να δημιουργήσετε ένα εικονικό partition μεγέθους έως 30GB. Είναι μια εικονική εγκατάσταση που δεν την προτείνω σε καμία περίπτωση. Καλύτερα χρησιμοποιείστε μια εικονική μηχανή (virtualbox) αν θέλετε να δείτε πως είναι το Ubuntu και να «δουλέψετε» κάπως μαζί του.

Προβλήματα του wubi

Κι εδώ ερχόμαστε στο προκείμενο. Στον τίτλο του άρθρου. Το wubi πλέον έχει σταματήσει την ανάπτυξή του. Είναι ένα παρωχημένο project. Είναι ένα project που σε σύγχρονους υπολογιστές με windows 8 , UEFI, GPT ..κλπ, θα προκαλέσει ζημιά. Μην το χρησιμοποιήσετε. Έχει αφαιρεθεί τελείως από την έκδοση Ubuntu 13.10 , ωστόσο υπάρχει ακόμη ως κομμάτι μέσα στο .iso. Πραγματικά απορώ γιατί υπάρχει ακόμη αυτό το project. Έπρεπε (κατά τη γνώμη μου) να έχει αφαιρεθεί ολοκληρωτικά από όλες τις σελίδες και τα .iso του Ubuntu ή να φτιάξουν τον κώδικα έτσι ώστε όταν εντοπίζει windows 8 και UEFI να μην συνεχίζει η εγκατάσταση. Τον κώδικα το βλέπω χλωμό να τον διορθώνουν, οπότε καλύτερα μην το δοκιμάσετε ποτέ. Υπάρχει βέβαια και η σχετική προειδοποίηση στην σελίδα του Ubuntu

Αφορμή γι’ αυτό το άρθρο ήταν ένα πρόσφατο πρόβλημα ενός χρήστη στο AskUbuntu. Ο άνθρωπος εγκατέστησε το Ubuntu 12.04 μέσω wubi σε μηχάνημα με windows 8 και UEFI και έγιναν όλα μπάχαλο. Ούτε τα windows δεν μπορεί να εκκινήσει πλέον. Υπάρχουν κι άλλα, πολλά τέτοια ζητήματα με τη συγκεκριμένη εφαρμογή.
