[Ubuntu 18.04] Thunderbird και Emoji

Υπερβολικά μεγάλα Emoji στο Subject line του Thunderbird και πως να το διορθώσετε

03 May 2018
Linux
ubuntu / desktop / thunderbird

Αναβαθμίζοντας στην έκδοση του Ubuntu 18.04 και χρησιμοποιώντας τον Thunderbird ως Email Client ίσως έρθετε αντιμέτωποι με αυτό το περίεργο σφάλμα, το οποίο δεν είμαι σίγουρος εάν οφείλεται στη γραμματοσειρά (.ttf) ή στον συνδυασμό γραμματοσειράς και Thunderbird.

Το πρόβλημα

Η νέα μόδα Marketing και Sales είναι η ενσωμάτωση εικονιδίων στο Subject Line του Email. Προσωπικά το βρίσκω αχρείαστο, αλλά δεν είναι απαραίτητα και κακό. Πολλές εταιρίες επιλέγουν αυτό τον τρόπο για να ενθαρρύνουν τον χρήστη έστω να ανοίξει το Email.
Πολλοί χρήστες όταν αντιληφθούν Sales/Marketing μηνύματα η πρώτη τους ενέργεια είναι να τα στείλουν στον κάδο χωρίς καν να τα ανοίξουν ή στον φάκελο με τα ανεπιθύμητα.
Στον Thunderbird, στο Ubuntu 18.04, αυτά τα εικονίδια εμφανίζονται υπερβολικά μεγάλα με αποτέλεσμα να χαλάει εκτός από την αιθητική και το subject line του μηνύματος.


1η Λύση

Ίσως σας φανεί απλή και λίγο ακατανόητη, αλλά η συγκεκριμένη λύση δούλεψε σε Debian (Buster). Στο Ubuntu 18.04 δούλεψε η δεύτερη(παρακάτω) λύση, εσείς όμως μπορείτε να δοκιμάσετε και τις δυο με τη σειρά.
Απλά μεταφέρουμε το αρχείο της γραμματοσειράς εκτός τοποθεσίας, κάνουμε μια ενημέρωση των fonts του συστήματος, έπειτα μεταφέρουμε ξανά το αρχείο γραμματοσειράς στη θέση του και κάνουμε ξανά ενημέρωση των Fonts του συστήματος. Τόσο απλό και επιτυγχάνεται με τις παρακάτω εντολές σε ένα τερματικό και φυσικά τον Thunderbird κλειστό.

sudo mv /usr/share/fonts/truetype/noto/NotoColorEmoji.ttf ~/
sudo fc-cache -v 
sudo mv NotoColorEmoji.ttf /usr/share/fonts/truetype/noto
sudo fc-cache -v

Αποτέλεσμα


2η Λύση

Η δεύτερη λύση έχει να κάνει με την αντικατάσταση της γραμματοσειράς με εκείνη που διατίθεται από την Google. Τη βρίσκουμε και την κάνουμε λήψη από το αποθετήριο
στο GitHub ⤏ https://github.com/googlei18n/noto-emoji
Εάν γνωρίζουμε τη βασική χρήση του GitHub μπορούμε να χρησιμοποιήσουμε τις γνώσεις μας για τη λήψη του αποθετηρίου. Διαφορετικά κάνουμε κλικ στο “Clone or Download” και κάνουμε λήψη το αρχείο σε μορφή Zip που από προεπιλογή θα μεταφερθεί στον κατάλογο Downloads (ή Λήψεις) του υπολογιστή μας.
Λαμβάνοντας υπόψη τη δεύτερη μέθοδο βλέπουμε παρακάτω την αντικατάσταση της γραμματοσειράς.

cd Downloads
unzip noto-emoji-master.zip
cd noto-emoji-master/fonts
sudo mv * /usr/share/fonts/truetype/noto/
sudo fc-cache -v
