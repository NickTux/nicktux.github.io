με το Boot Repair

03 Jul 2013
Ubuntu / Linux
troubleshooting / repair / boot / over_a_year_old

Το boot-repair είναι ένα πρόγραμμα που θα σας βοηθήσει να διορθώσετε τυχών προβλήματα εκκίνησης του λειτουργικού συστήματος Ubuntu. Τέτοια προβλήματα μπορεί να είναι:

  • Δεν μπορώ να κάνω boot στα Windows αφού εγκατέστησα το Ubuntu
  • Επιλέγω την εισαγωγή Ubuntu αλλά μου εμφανίζει μια μαύρη οθόνη που γράφει "grub-rescue"
  • Ο grub-bootloader δεν εμφανίζεται καθόλου
  • Βλέπω μόνο μια μαύρη οθόνη και τίποτε άλλο
  • Πρόβλημα εκκίνησης σε UEFI/Secure Boot σύστημα
ΑΥΤΗ Η ΔΗΜΟΣΙΕΥΣΗ ΕΧΕΙ ΓΙΝΕΙ ΠΑΝΩ ΑΠΟ ΕΝΑ ΧΡΟΝΟ ΠΡΙΝ! ΜΠΟΡΕΙ ΝΑ ΜΗΝ ΙΣΧΥΕΙ ΠΛΕΟΝ.

Τέτοια ζητήματα μπορεί να προκύψουν μετά την εγκατάσταση ενός νέου λειτουργικού συστήματος παράλληλα με το υπάρχον, είτε αυτό είναι τα Windows μετά το Ubuntu , είτε κάποιο άλλο λειτουργικό σύστημα Linux.

Απλή διόρθωση

Συνδέστε τον υπολογιστή σας στο Internet με ένα καλώδιο Ethernet. Εισάγετε το CD ή Usb που έχετε φτιάξει και κάνετε boot από αυτό. Όταν εγκαταστήσατε το Ubuntu για πρώτη φορά , πιθανότατα το κάνατε μέσω κάποιου CD ή Usb (εκτός της περίπτωσης wubi.exe) , αυτά ονομάζονται και Live CD ή Live Usb. Εισάγετε στον υπολογιστή σας αυτό το Live CD ή Usb και ρυθμίστε τον έτσι ώστε να κάνει boot από αυτό. Όπως όταν εγκαταστήσατε το Ubuntu.

Προσοχή: Η έκδοση Ubuntu πρέπει να είναι μια υποστηριζόμενη έκδοση και όχι EOL (End Of Life).

Πατήστε , αν σας ζητηθεί , Try Ubuntu (Δοκιμή). Δεν θέλουμε να εγκαταστήσουμε ξανά το Ubuntu. Έπειτα ανοίξτε ένα τερματικό με τον συνδυασμό πλήκτρων (CTRL+ALT+T) και εκτελέστε τις παρακάτω εντολές μια-μια.

sudo add-apt-repository ppa:yannubuntu/boot-repair &amp;&amp; sudo apt-get update
sudo apt-get install -y boot-repair &amp;&amp; boot-repair

Αυτό που θα κάνουν οι παραπάνω εντολές είναι να εγκαταστήσουν την εφαρμογή boot-repair και να την τρέξουν. Τελειώνοντας την εκτέλεση τους θα δείτε να ανοίγει ένα παράθυρο σαν το παρακάτω

Πατήστε το Recommended repair για να προσπαθήσει το πρόγραμμα να διορθώσει λάθη που θα βρει στην εκκίνηση του υπολογιστή σας.

Σημαντικό: Όταν τελειώσει η διόρθωση , τότε το πρόγραμμα θα σας δώσει ένα URL με διάφορα στοιχεία. Αυτό το URL κρατήστε το και σε περίπτωση που δεν διορθωθεί η εκκίνηση του υπολογιστή σας, δείξτε το σε ένα φόρουμ υποστήριξης για περαιτέρω βοήθεια.
